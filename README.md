# Fn-Logik-Mathematik
Die Mathematik des Prozesses: Berechnungen ohne Multiplikation, nur durch Addition. Fn-Logik integriert Zeit als Relativität direkt in den Rechenweg – einfach, hardwarenah und ohne Differenzialgleichungen.
