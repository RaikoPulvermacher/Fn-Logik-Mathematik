# Die Basis der Fn-Logik

In der Fn-Logik existiert keine statische Multiplikation. Alles ist ein fortlaufender Prozess der Addition, bei dem der Faktor Zeit die Entwicklung der Werte bestimmt.

## Die fundamentale Prozesskette
$fn_1 + fn_1 = fn_2$
$fn_2 + fn_1 = fn_3$
$fn_3 + fn_2 = fn_5$
$fn_5 + fn_3 = fn_8$
$fn_8 + fn_5 = fn_{13}$

## Anwendung: m/s (Geschwindigkeit) und Strecke
Der physikalische Weg wird hier nicht durch eine Formel berechnet, sondern "erwandert" durch die Addition der Zustände:

| Schritt | Logik-Prozess | Resultat |
| :--- | :--- | :--- |
| 1 | $fn_1 + fn_1$ | $fn_2$ |
| 2 | $fn_2 + fn_1$ | $fn_3$ |
| 3 | $fn_3 + fn_2$ | $fn_5$ |
| 4 | $fn_5 + fn_3$ | $fn_8$ |
| 5 | $fn_8 + fn_5$ | $fn_{13}$ |

**Kernbotschaft:** Mathematik ist Zeit. Jeder Schritt ist ein Takt des Prozessors.
